Для информации:
1) Логи от программ "DAVrun" и "SERVICE-DAVrun" создаются в папке: C:\ProgramData\DAVrun.
2) Временная папка для развёртывания самораспаковывающего архива создаётся по пути: C:\ProgramData\DAVrun\Temp$DEV$RUN.
3) Программа "NamePC Full" при запуске создаёт в этой же папке файл с полным именем компьютера.
4)Программа "NamePC Cut-7" при запуске создаёт в этой же папке файл с обрезанным именем компьютера (обрезаются первые 7 символов).
Программы NamePC могут пригодится для быстрого создания файла с именем компьютера (можно запускать с любого места на компьютере).
